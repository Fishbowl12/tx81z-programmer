The Snapshot Menu

This menu is used to save and load your snapshots from files.

New creates an empty snapshot with no file name.

Open loads a snapshot into the program from a file.

Save saves the snapshot to the file indicated in the file name indicator. If the snapshot has no file name, you will be prompted for one.

Save As... saves the snapshot in a new file.

Save Copy As... saves the snapshot in a new file, but doesn't change the name that the current working snapshot is filed under. This is useful for making backup copies of the snapshot.

Load Copy loads a snapshot file into the current snapshot, but preserves the current file name. If the snapshot file that you're loading is incomplete, and you already have a file open, then the items that the file does contain will be overlaid onto the current snapshot.
